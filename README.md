# hidden-voices

Generated outputs and audio renderings for the project *Hidden Voices:
Generating Counterpoint from Acoustic First Principles*, completed as
Independent Work for the Program in Applied and Computational
Mathematics at Princeton, advised by Professor Bernard Chazelle.

The project asks whether an algorithm guided by a dissonance model
derived from acoustic physics can reconstruct a musically coherent
hidden second voice for Bach's Cello Suites (BWV 1007–1012), which
are written for a solo instrument but widely believed to imply
polyphonic structure.

## Contents

- `prelude/` — generated hidden voice for Suite No. 1 Prelude
  (score PDF and MP3 rendering)
- `sarabande/` — generated hidden voice for Suite No. 1 Sarabande
  (score PDF and MP3 rendering)
- `minuet/` — generated hidden voice for Suite No. 1 Minuet I/II
  (score PDF and MP3 rendering)

The Prelude is included as a representative case of the algorithm's
limitations on arpeggiated textures, discussed in the report.

Audio was rendered from the MusicXML output using MuseScore 4.
